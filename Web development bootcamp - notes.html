Block elements

A block element will take up horizontal space (100% width). Paragraphs < p > and divs < div > are block elements.

Block elements:

  • Paragraph
  • Divs

Inline elements

An inline element takes up vertical space and can be placed in a container, like a < div >. You can place an inline element, such as a span < span > or an anchor inside a block element.

A div can be used to group different elements and style them using CSS. You can also use a span to add CSS styling. For example: when you want to highlight important words (without using the bold tag). I'm gonna fix that later. After learning CSS.

Inline elements

  • Span
  • Anchor
  • Image

An Odd Assortment of Elements

There's some more HTML elements that might come in handy:

  • Horizontal Rule < hr >
  • Line Break < br >
  • Superscript < sup >
  • Subscript < sub >

Horizontal Rule

A Horizontal Rule can be used to divide different sections on a page. Like you see on this page.

Line Break

You can use a Line Break to force a break. For example: you can use a line break when you're publishing a poem and want to control where each line breaks.

Superscript

You can use Superscript to add references to your website. You can use nested elements to add an anchor to a superscript, like you see on Wikipedia.

Subscript

You can also use the Subscript, even though it's not used often. You can use it to make fractions.

Entity Codes

You can use Entity Codes to display characters that normally would be invalid. For example: the copyright and trademark symbol.

There's also reserved characters in HTML. These are the characters that we use to write our HTML code. These reserved characters may cause issues in Visual Studio Code. This is why we use entity codes. You can find all entity codes on this website
